# Combat System Sample
A simple combat system. 10 characters have weapons. Battle is simulated, and the winner is announced alongside a scoreboard.

## Technologies Used
* Development Environment
    * Unity 2020.3.6f1
    * VSCodium
    * Atom
    * macOS 11.2.2
* Dependencies
    * TextMeshPro 3.0.5
